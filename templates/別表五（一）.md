## Ⅰ 利益積立金額の計算（ICB00000）

### 繰越損益金（ICB00400）

| タグID | 欄 | 値 |
|--------|----|----|
| ICB00410 | 期首残高 | |
| ICB00430 | 減 | |
| ICB00440 | 増 | |
| ICB00450 | 期末残高 | |

> ICB00410: 前期末残高（前期の ICB00450）。第1期は0。
> ICB00430: 期首残高と同値（振替）。
> ICB00440: ICB00410 ＋ 当期純利益（別表四 ARV）。損失の場合はマイナス値。
> ICB00450: ICB00440と同値。

---

### 未納法人税及び地方法人税（ICB00530）

| タグID | 欄 | 値 |
|--------|----|----|
| ICB00540 | 期首残高 | |
| ICB00590 | 増（当期確定） | |
| ICB00610 | 期末残高 | |

> ICB00540: 前期末の未納額（マイナス値）。第1期は0。
> ICB00590: −(確定法人税額 + 確定地方法人税額)（マイナス値）。別表一 BGB00210 + BGC00140。
> ICB00610: ICB00540 ＋ ICB00590（マイナス値）。

---

### 未納道府県民税（ICB00620）

| タグID | 欄 | 値 |
|--------|----|----|
| ICB00630 | 期首残高 | |
| ICB00680 | 増（当期確定） | |
| ICB00700 | 期末残高 | |

> ICB00630: 前期末の未納額（マイナス値）。第1期は0。
> ICB00680: −(均等割＋法人税割)（マイナス値）。
> ICB00700: ICB00630 ＋ ICB00680（マイナス値）。

---

### 差引合計額（ICB00800）

| タグID | 欄 | 値 |
|--------|----|----|
| ICB00810 | 期首残高 | |
| ICB00830 | 減 | |
| ICB00840 | 増 | |
| ICB00850 | 期末残高 | |

> ICB00810 = ICB00410 ＋ ICB00540 ＋ ICB00630（合計）
> ICB00830 = ICB00430と同値
> ICB00840 = ICB00440 ＋ ICB00590 ＋ ICB00680（合計）
> ICB00850 = ICB00450 ＋ ICB00610 ＋ ICB00700（合計）

## XTX生成プロンプト（AIへ）

上記の記入欄に入力された値をもとに、別表五(一)（HOA511）のXML断片をPython辞書形式で出力してください。

### 基本ルール

- バージョンは frontmatter の `xtx_version` を使用する
- マイナス値はそのまま負の整数で出力する（例: `-159400`）
- 値が0または未記入の項目は省略可（差引合計等の集計行は省略しない）
- REFタグは文字列でそのまま出力する

### タグ構造

```python
{
  "ICA00000": {
    "ICA00010": {
      "ICA00020": "REF:JIGYO_NENDO_FROM",
      "ICA00030": "REF:JIGYO_NENDO_TO"
    },
    "ICA00040": "REF:NOZEISHA_NM"
  },
  "ICB00000": {
    "ICB00400": {                            # 繰越損益金
      "ICB00410": <期首残高>,
      "ICB00420": {
        "ICB00430": <減（期首と同値）>,
        "ICB00440": <増（= 期首 + 当期純利益）>
      },
      "ICB00450": <期末残高（= ICB00440）>
    },
    "ICB00520": {                            # 未納法人税等コンテナ
      "ICB00530": {                          # 未納法人税及び地方法人税
        "ICB00540": <期首（マイナス値）>,
        "ICB00550": {
          "ICB00570": {
            "ICB00590": <当期確定（マイナス値）>  # -(法人税+地方法人税)
          }
        },
        "ICB00610": <期末（マイナス値）>      # ICB00540 + ICB00590
      },
      "ICB00620": {                          # 未納道府県民税
        "ICB00630": <期首（マイナス値）>,
        "ICB00640": {
          "ICB00660": {
            "ICB00680": <当期確定（マイナス値）>  # -(均等割+法人税割)
          }
        },
        "ICB00700": <期末（マイナス値）>      # ICB00630 + ICB00680
      }
    },
    "ICB00800": {                            # 差引合計額
      "ICB00810": <期首残高合計>,
      "ICB00820": {
        "ICB00830": <減（ICB00430と同値）>,
        "ICB00840": <増合計（ICB00440+ICB00590+ICB00680）>
      },
      "ICB00850": <期末残高合計（ICB00450+ICB00610+ICB00700）>
    }
  },
  "ICC00000": {
    "ICC00010": {                            # 資本金又は出資金
      "ICC00020": <期首残高>,
      "ICC00030": {"ICC00050": 0},           # 増（増資なしは0）
      "ICC00060": <期末残高>
    },
    "ICC00200": {                            # 資本金等合計
      "ICC00210": <期首残高>,
      "ICC00220": {"ICC00240": 0},           # 増（増資なしは0）
      "ICC00250": <期末残高>
    }
  }
}
```

### 計算順序

1. `ICB00440` = ICB00410 ＋ 当期純利益（別表四ARV）。損失はマイナス値。
2. `ICB00450` = ICB00440
3. `ICB00590` = −(確定法人税額 + 確定地方法人税額)
4. `ICB00610` = ICB00540 + ICB00590
5. `ICB00680` = −(道府県民税確定額)
6. `ICB00700` = ICB00630 + ICB00680
7. `ICB00810` = ICB00410 + ICB00540 + ICB00630
8. `ICB00840` = ICB00440 + ICB00590 + ICB00680
9. `ICB00850` = ICB00450 + ICB00610 + ICB00700

### 注意事項

- 第1期（創業初年度）: ICB00410/ICB00540/ICB00630 はすべて0
- ICB00550/ICB00640 は複合型コンテナ。ICB00590/ICB00680 はその孫要素。
- 未納道府県民税が0の場合は ICB00620 を省略可
- 資本金の増減がない場合、ICC00030/ICC00220 の増欄（値=0）は省略可
