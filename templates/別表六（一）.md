# 別表六(一)　所得税額の控除に関する明細書

> 受取利子・配当等に係る源泉所得税を法人税額から控除する。控除額を別表一（BGB00180）に転記する。
> **源泉所得税の控除がない場合はこの別表を省略できる。**

---

| FZA00030 | 事業年度（至） | |
| FZA00040 | 法人名 | |

---

## 預貯金・公社債利子等の源泉税（FZC00010）

> 銀行預金利子・公社債利子等に係る源泉所得税（15.315%）を記入する。

| タグID | 項目 | 値 |
|--------|------|----|
| FZC00020 | 収入金額（円） | |
| FZC00030 | 課される所得税額（円）（収入 × 15.315%） | |
| FZC00040 | 控除を受ける所得税額（円） | |

> 通常 FZC00040 = FZC00030（全額控除）。

---

## 計（FZC00300）

| タグID | 項目 | 値 |
|--------|------|----|
| FZC00310 | 収入金額 合計（円） | |
| FZC00320 | 課される所得税額 合計（円） | |
| FZC00330 | 控除を受ける所得税額 合計（円）（本書き） | |

> FZC00330 = FZC00040 の合計（FZC00010 のみの場合は FZC00040 と同値）。
> FZC00330 の値を別表一 BGB00180（所得税額の控除）に転記する。

## XTX生成プロンプト（AIへ）

上記の記入欄に入力された値をもとに、別表六(一)（HOB016）のXML断片をPython辞書形式で出力してください。

### 基本ルール

- バージョンは `6.0` を使用する（固定）
- 値が0または未記入の項目は省略する
- REFタグは文字列でそのまま出力する
- **源泉所得税が0円の場合はこの別表を出力しない**

### タグ構造

```python
{
  "FZA00000": {
    "FZA00010": {
      "FZA00020": "REF:JIGYO_NENDO_FROM",
      "FZA00030": "REF:JIGYO_NENDO_TO"
    },
    "FZA00040": "REF:NOZEISHA_NM"
  },
  # FZC00000: 種別ごとの源泉税明細（XSD順序: FZC00010→FZC00050→FZC00090→FZC00220→FZC00260→FZC00300）
  "FZC00000": {
    "FZC00010": {                         # 預貯金・公社債利子等
      "FZC00020": <収入金額>,
      "FZC00030": <課される所得税額>,
      "FZC00040": <控除を受ける所得税額>  # 通常 FZC00030 と同値
    },
    # FZC00050: 剰余金の配当等（配当所得がある場合のみ）
    # "FZC00050": {
    #   "FZC00060": <収入金額>,
    #   "FZC00070": <課される所得税額>,
    #   "FZC00080": <控除を受ける所得税額>
    # },
    "FZC00300": {                         # 計（必須）
      "FZC00310": <収入金額合計>,
      "FZC00320": <課される所得税額合計>,
      "FZC00330": <控除を受ける所得税額合計>  # 別表一BGB00180に転記
    }
  },
  # FZG00000: その他（FZC以外の源泉税がある場合のみ）
  "FZG00000": {
    "FZG00010": [                         # リスト形式（最大5件）
      {
        "FZG00020": "<支払者の氏名又は法人名>",
        "FZG00030": "<支払者の住所又は所在地>",
        "FZG00040": "<支払を受けた年月日（gen:yymmdd形式: '5,07,08,31'）>",
        "FZG00050": <収入金額>,
        "FZG00060": <控除を受ける所得税額>
      },
      # ... 2件目以降
    ],
    "FZG00080": {                         # 計
      "FZG00090": <収入金額合計>,
      "FZG00100": <控除を受ける所得税額合計>
    }
  }
}
```

### 計算順序

1. `FZC00030` = FZC00020 × 15.315%（1円未満切捨）
2. `FZC00040` = FZC00030（全額控除の場合。一部控除の場合は別途計算）
3. `FZC00310` = FZC00020（FZC00010のみの場合は同値）
4. `FZC00320` = FZC00030
5. `FZC00330` = FZC00040（これが別表一 BGB00180 の値）

### 別表一（HOA112）への転記

HOB016 を使用する場合、BGB00000 に `BGB00180` を追加し、`BGB00190`・`BGB00210` を再計算する：

```python
"BGB00000": {
    "BGB00010": 所得,
    "BGB00020": 法人税額,
    "BGB00120": {"BGB00150": 法人税額},
    "BGB00180": 所得税控除額,               # ← FZC00330 の値
    "BGB00190": 法人税額 - 所得税控除額,    # 差引確定税額
    "BGB00210": 法人税額 - 所得税控除額,    # 差引納付税額（中間納付なしの場合）
}
```

### 注意事項

- FZC00330 は「本書き」タグ（内書き FZC00327 との混同に注意）
- FZG00040（支払を受けた年月日）は gen:yymmdd 形式: `"5,07,08,31"` のような文字列
- FZG00010 は複数ある場合リスト形式で渡す
- FZC00000 に FZC00010 以外の種別（FZC00050 等）を追加する場合も FZC00300 の合計に含める
