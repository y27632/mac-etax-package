# 別表七(一)　欠損金の損金算入等に関する明細書

> 青色申告法人が前期以前の繰越欠損金を当期所得から控除する。控除額を別表四（ART00010）に転記する。
> **繰越欠損金がない場合はこの別表を省略できる。**

---

|--------|------|----|
| MCA00020 | 事業年度（自） | |
| MCA00030 | 事業年度（至） | |
| MCA00040 | 法人名 | |

## 控除前所得金額・損金算入限度額（MCB00002/MCB00004）

| タグID | 項目 | 値 |
|--------|------|----|
| MCB00002 | 控除前所得金額（円）（= 別表四 ARS50010） | |
| MCB00007 | 区分（kubun_CD） | `1`=100%（中小法人等） / `5`=50%（大法人） |
| MCB00008 | 損金算入限度額（円） | |

> 中小法人等（資本金1億円以下等）: MCB00008 = MCB00002（100%控除可）。kubun_CD = "1"。
> 大法人: MCB00008 = MCB00002 × 50%。kubun_CD = "5"。

---

## 繰越欠損金明細（先頭行）（MCB00010）

> 繰越欠損金が1件のみ、かつ当期で全額控除する場合（翌期繰越なし）に MCB00010 を使用する。

| タグID | 項目 | 値 |
|--------|------|----|
| MCB00030 | 欠損事業年度（自）（gen:yymmdd 形式） | |
| MCB00040 | 欠損事業年度（至）（gen:yymmdd 形式） | |
| MCB00060 | 区分（青色欠損 kubun_CD="1"） | `1` |
| MCB00090 | 控除未済欠損金額（円）（その期の欠損金の残額） | |
| MCB00100 | 当期控除額（円） | |

> MCB00010 に翌期繰越額（MCB00210相当）は存在しない。当期で使い切る前提の行。

---

### MCB00010 と MCB00110 の使い分け

- **MCB00010**（先頭行専用）: 翌期繰越額タグが存在しない。当期で全額控除する場合のみ使用。
- **MCB00110**（2行目以降・リスト）: 翌期繰越額（MCB00210）タグあり。翌期繰越が生じる行はこちら。
- 欠損金が1件でも翌期繰越が生じる場合は MCB00110 のみ使用（MCB00010 は省略）。
- 欠損金が2件以上ある場合: MCB00010 に先頭行、MCB00110 に2件目以降をリスト形式で追加。

### 計算順序

1. 古い期の欠損金から順に控除する（先入先出し）
2. 各行の `翌期繰越額 = 控除未済欠損金額 - 当期控除額`
3. `MCB00230` = 各行の控除未済欠損金額の合計
4. `MCB00240` = 各行の当期控除額の合計（≦ MCB00008）
5. `MCB00250` = 各行の翌期繰越額の合計
6. `MCB00370` = MCB00250 + MCB00270（当期欠損翌期繰越分。当期黒字なら MCB00250 と同値）

### 別表四（HOA420）との対応

MCB00240（当期控除額合計）= 別表四 ART00010

```python
# 別表四 ART セクションに欠損金控除を追加
"ART00000": {
    "ART00010": <MCB00240>,  # 欠損金等の当期控除額
}
# 課税所得 = 別表四 ARV00010（ARS50010 - ART00010）
```

### 注意事項

- MCB00030/MCB00040 等の日付は gen:yymmdd 形式（`"5,06,02,01"` のような文字列）
- kubun_CD は文字列: `"1"`（青色欠損）
- MCB00110 が複数ある場合はリスト形式: `"MCB00110": [{...}, {...}]`
- MCB00010 の MCB00050 は `{"MCB00060": {"kubun_CD": "1"}}` の構造
- MCB00110 の MCB00150 は `{"MCB00160": {"kubun_CD": "1"}}` の構造
