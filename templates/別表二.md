## ① 同族会社の判定（VAB00000）

| タグID | 項目 | 値 |
|--------|------|----|
| VAB00010 | 期末現在の出資の総額（円） | |
| VAB00020 | 上位3順位の出資額合計（円） | |
| VAB00030 | 出資の金額による判定（%） | |
| VAB00090 | 期末現在の社員の総数（名） | |
| VAB00100 | 社員3人以下＋同族関係者の最多数（名） | |
| VAB00110 | 社員の数による判定（%） | |
| VAB00120 | 同族会社の判定割合（%） | |

> VAB00030 = VAB00020 ÷ VAB00010 × 100（整数）
> VAB00110 = VAB00100 ÷ VAB00090 × 100（整数）
> VAB00120 = VAB00030・VAB00110 のうち最高値

## XTX生成プロンプト（AIへ）

上記の記入欄に入力された値をもとに、別表二（HOA201 v4.0）のXML断片をPython辞書形式で出力してください。

### 基本ルール

- バージョンは `4.0` を使用する（固定）
- 値が0または未記入の項目は省略する
- 判定割合はすべて整数（小数切捨て）で出力する
- `kubun_CD` は文字列で出力する

### タグ構造

```python
{
  "VAB00000": {
    "VAB00010": <出資の総額（円）>,
    "VAB00020": <上位3順位の出資額合計（円）>,
    "VAB00030": <出資金額による判定割合（%・整数）>,
    "VAB00090": <社員の総数（名）>,
    "VAB00100": <社員3人以下+同族関係者の最多数（名）>,
    "VAB00110": <社員数による判定割合（%・整数）>,
    "VAB00120": <同族会社の判定割合（%・整数）>
  },
  "VAC00000": {
    "VAC00010": <上位1順位の出資額（円）>,
    "VAC00020": <出資金額による判定割合（%・整数）>,
    "VAC00050": <社員1人+同族関係者の最多数（名）>,
    "VAC00060": <社員数による判定割合（%・整数）>,
    "VAC00070": <特定同族会社の判定割合（%・整数）>
  },
  "VAD00000": {"kubun_CD": "<1=同族会社 / 2=非同族会社>"},
  "VAE00000": {                        # 単一要素（リストではない）
    "VAE00010": {                      # 本人（1人目の社員）
      "VAE00020": {                    # 順位
        "VAE00030": <出資額順位（整数）>,
        "VAE00040": <議決権数順位（整数）>,
      },
      "VAE00050": {                    # 判定基準の株主(社員)等
        "VAE00060": "<住所>",
        "VAE00070": "<氏名>",
      },
      "VAE00080": {                    # 株式数/出資の金額等
        "VAE00120": {                  # その他の株主等
          "VAE00130": <出資額（円）>,
          "VAE00140": {                # 議決権の数
            "VAE00160": <議決権の数（整数）>,
          },
        },
      },
    },
    "VAE00170": [                      # 2人目以降（maxOccurs=12）
      {
        "VAE00180": {                  # 順位
          "VAE00190": <出資額順位（整数）>,
          "VAE00200": <議決権数順位（整数）>,
        },
        "VAE00210": {                  # 判定基準の株主(社員)等
          "VAE00220": "<住所>",
          "VAE00230": "<氏名>",
        },
        "VAE00250": {                  # 株式数/出資の金額等
          "VAE00290": {                # その他の株主等
            "VAE00300": <出資額（円）>,
            "VAE00310": {              # 議決権の数
              "VAE00330": <議決権の数（整数）>,
            },
          },
        },
      }
      # 社員が4人以上いる場合はオブジェクトを追加（最大12エントリ）
    ],
  }
}
```

### 注意事項

- VAE00000 は単一要素（リストではない）。1人目は `VAE00010`、2人目以降は `VAE00170`（maxOccurs=12）に入れる。
- VAE00130（出資額）は VAE00080 > VAE00120 の中に入れる（複合型コンテナ）。
- VAE00300（2人目以降の出資額）は VAE00250 > VAE00290 の中に入れる。
- VAB00030/VAB00110/VAB00120 等の判定割合タグは simpleContent 型 → 整数値を直接渡す。
- 合同会社の場合、「株式数」=「出資の金額」、「株主」=「社員」として扱う。
- バージョンは `5.0` を使用する（`4.0` は旧バージョン・RHO0022 v24.0.2 では5.0が必須）。
