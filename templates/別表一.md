## ① 法人税額（BGB00000）

| タグID | 項目 | 値 |
|--------|------|----|
| BGB00010 | 所得金額（円） | |
| BGB00020 | 法人税額（円） | |
| BGB00150 | 控除前税額（円） | |
| BGB00180 | 控除税額・源泉所得税等（円） | |
| BGB00190 | 差引法人税額（円） | |
| BGB00210 | 確定法人税額（円） | |
| BGB00460 | 欠損金等の当期控除額（円） | |
| BGB00470 | 欠損金額（損失の場合のみ・円） | |

> BGB00010: 別表四 ARV（所得金額）から転記
> BGB00020: 別表一（次葉）QHB00100 から転記
> BGB00150: BGB00020と同値（中間納付なしの場合）
> BGB00190: BGB00150 − BGB00180
> BGB00210: BGB00190の1,000円未満切捨
> BGB00460: 白色申告は0。損失の場合は BGB00470 に欠損金額を記入し BGB00210=0。

### 源泉所得税がある場合のみ（BGB00220）

| BGB00260 | 控除した金額（円） | |
| BGB00270 | 控除しきれなかった金額（円） | |

> 受取利息等で源泉税がない場合は BGB00180/BGB00220 を省略。

## XTX生成プロンプト（AIへ）

上記の記入欄に入力された値をもとに、別表一（HOA112）のXML断片をPython辞書形式で出力してください。

### 基本ルール

- バージョンは frontmatter の `xtx_version` を使用する
- 値が0または未記入の項目は辞書から省略する（ただし BGA00210 配下のチェック欄は全項目必須）
- `REF:NOZEISHA_NM` 等のREFタグは文字列でそのまま出力する
- `kubun_CD` は文字列で出力する（例: `{"kubun_CD": "1"}`）

### タグ構造

```python
{
  "BGA00000": {
    "BGA00010": "REF:TEISYUTSU_DAY",
    "BGA00020": "REF:ZEIMUSHO",
    "BGA00030": "REF:NOZEISHA_ADR",
    "BGA00040": "REF:NOZEISHA_TEL",
    "BGA00050": {
      "BGA00060": "REF:NOZEISHA_NM_KN",
      "BGA00070": "REF:NOZEISHA_NM"
    },
    "BGA00080": "REF:NOZEISHA_BANGO",
    "BGA00090": {
      "BGA00100": "REF:DAIHYO_NM_KN",
      "BGA00110": "REF:DAIHYO_NM"
    },
    "BGA00120": "REF:DAIHYO_ADR",
    "BGA00130": {"kubun_CD": "1"},             # 普通法人
    "BGA00140": "REF:JIGYO_NAIYO",
    "BGA00150": "REF:SHIHON_KIN",
    "BGA00170": {"kubun_CD": "<同族:1/非同族:2>"},
    "BGA00210": {                              # チェック欄（全項目出力）
      "BGA00220": {"kubun_CD": "1"},
      "BGA00230": {"kubun_CD": "1"},
      "BGA00240": {"kubun_CD": "1"},
      "BGA00250": {"kubun_CD": "1"},
      "BGA00260": {"kubun_CD": "1"}
    },
    "BGA00290": <売上金額（円）>,
    "BGA00330": {
      "BGA00340": "REF:JIGYO_NENDO_FROM",
      "BGA00350": "REF:JIGYO_NENDO_TO"
    },
    "BGA00410": "<決算確定の日>"
  },
  "BGB00000": {
    "BGB00010": <所得金額（円）>,              # 別表四から転記
    "BGB00020": <法人税額（円）>,              # 別表一（次葉）から転記
    "BGB00120": {
      "BGB00150": <控除前税額（円）>           # BGB00020と同値
    },
    # 源泉税がある場合のみ:
    # "BGB00180": <控除税額（源泉所得税等）>,
    "BGB00190": <差引法人税額（円）>,          # BGB00150 - BGB00180
    "BGB00210": <確定法人税額（円）>,          # 1,000円未満切捨
    # 源泉税がある場合のみ:
    # "BGB00220": {
    #   "BGB00230": <所得税の額>,
    #   "BGB00250": <計>,
    #   "BGB00260": <控除した金額>,
    #   "BGB00270": <控除しきれなかった金額>
    # },
    "BGB00460": 0                              # 白色申告=0 / 青色申告は欠損金控除額
    # 損失の場合のみ: "BGB00470": <欠損金額>
  },
  "BGC00000": {
    "BGC00010": {
      "BGC00020": {
        "BGC00030": <法人税額（BGB00210と同値）>
      },
      "BGC00050": <課税標準法人税額（BGC00030と同値）>
    },
    "BGC00060": <地方法人税額（BGC00050 × 10.3%）>,
    "BGC00080": <所得地方法人税額（BGC00060と同値）>,
    "BGC00120": <差引地方法人税額（BGC00080と同値）>,
    "BGC00140": <確定地方法人税額（100円未満切捨）>,
    "BGC00150": {                              # 還付金額（中間納付なし=0）
      "BGC00260": {"BGC00280": 0}
    }
  }
}
```

### 計算順序

1. `BGB00190` = BGB00150 − BGB00180（源泉税なしなら BGB00150と同値）
2. `BGB00210` = BGB00190 の1,000円未満切捨
3. `BGC00060` = BGB00210 × 10.3%（1円未満切捨）
4. `BGC00140` = BGC00060 の100円未満切捨

### 注意事項

- 損失の場合: BGB00470 に欠損金額を記入し、BGB00210=0 とする
- 源泉税がない場合: BGB00180/BGB00220 を省略し、BGB00190=BGB00150
- 中間申告がある場合: BGB00200（中間納付額）を追加する
