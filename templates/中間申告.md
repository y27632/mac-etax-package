# 中間申告（予定申告）　別表十九

> 前事業年度の確定法人税額が20万円超の法人は、事業年度開始から6ヶ月経過後2ヶ月以内に**中間申告**が必要。
> 「予定申告」は前年度税額 × 6/12 を申告・納付する最もシンプルな方式。
> **仮決算による中間申告**（実際の中間期決算に基づく申告）は対象外（別手続き）。

---

## 中間申告の要否確認

| 条件 | 中間申告 |
|------|---------|
| 前年度確定法人税額 ≤ 200,000円 | 不要 |
| 前年度確定法人税額 > 200,000円 | **必要** |
| 設立第1期（前年度なし） | 不要 |

> 前年度確定法人税額は前年の別表一（BGB00020「法人税額」）を参照。

## 法人税額の計算（RZD00000）

| タグID | 項目 | 値 |
|--------|------|----|
| RZD00030 | 前事業年度の法人税額 | |
| RZD00040 | 同上のうち土地譲渡税額等及び税額控除超過額相当額等の加算額 | |
| RZD00050 | 差引法人税額（RZD00030 − RZD00040） | |
| RZD00055 | 月数換算（前事業年度の月数。通常12） | |
| RZD00070 | 納付すべき法人税額（RZD00050 × 6 ÷ RZD00055、百円未満切り捨て） | |

> RZD00040（土地譲渡税額等の加算額）は通常0。記入不要な場合は省略。

---

## 地方法人税額の計算（RZE00000）

| タグID | 項目 | 値 |
|--------|------|----|
| RZE00030 | 前課税事業年度の地方法人税額 | |
| RZE00040 | 同上のうち税額控除超過額相当額の加算額等 | |
| RZE00050 | 差引地方法人税額（RZE00030 − RZE00040） | |
| RZE00055 | 月数換算（前課税事業年度の月数。通常12） | |
| RZE00070 | 納付すべき地方法人税額（RZE00050 × 6 ÷ RZE00055、百円未満切り捨て） | |

> RZE00040は通常0。地方法人税 = 法人税額 × 10.3%（別表一を参照）。

---

## 記入例（計算）

```
前年度法人税額:  500,000円（BGB00020）
前年度地方法人税額: 51,500円（BGB00300 = 500,000 × 10.3%）

法人税中間納付額:   500,000 × 6 / 12 = 250,000円
地方法人税中間納付額: 51,500 × 6 / 12 = 25,700円（百円未満切り捨て）
```

### 計算式

```python
# 法人税中間納付額（百円未満切り捨て）
rzd00070 = (rzd00050 * 6 // rzd00055 // 100) * 100

# 地方法人税中間納付額（百円未満切り捨て）
rze00070 = (rze00050 * 6 // rze00055 // 100) * 100
```

---

## 注意事項

- 修正申告の場合のみ `RZB00000`（この申告前の税額）を記入する。通常は省略
- `RZF00000`（適用事業年度等）は通常省略
- 申告・納付の期限: 事業年度開始から8ヶ月以内（6ヶ月経過後2ヶ月以内）
