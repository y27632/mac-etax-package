## 別表間の依存関係

XTX生成スクリプトを組み立てる際は、以下の転記関係に従って値を設定すること。

```
別表七（一）
  └─[当期控除額合計 MCB00240]──→ 別表四 ART00010（欠損金の損金算入額）

別表十五
  └─[損金不算入額 EGE00000]────→ 別表四 ARG00010（交際費等の損金不算入額）

別表六（一）
  └─[控除所得税額 FZC00330等]──→ 別表一 BGB00180（源泉徴収税額）

別表四
  └─[所得金額 ARU00170]────────→ 別表一 BGB00010（所得の金額）

別表一（次葉）
  └─[法人税額合計 QHC00010]────→ 別表一 BGB00020（所得に対する法人税額）
  └─[地方法人税額 QHC00050]────→ 別表一 BGC00060（地方法人税額）

別表一
  └─[確定法人税額 BGB00020]────→ 別表五（二）納付税額欄
```

## 推奨記入・生成順序

```
1. 別表十五（交際費がある場合）
2. 別表七（一）（繰越欠損金がある場合）
3. 別表四
4. 別表二
5. 別表六（一）（源泉所得税がある場合）
6. 別表一 → 別表一（次葉）
7. 別表五（一）↔ 別表五（二）（相互参照）
8. 法人事業概況説明書
```

XTXへの `add_form()` 呼び出し順序は記入順序とは異なる。XSD sequence 定義に従った順序については `KNOWHOW.md` §10を参照。
