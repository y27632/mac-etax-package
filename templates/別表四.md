## XTX生成プロンプト（AIへ）

上記の記入欄に入力された値をもとに、別表四（HOA420）のXML断片をPython辞書形式で出力してください。

### 基本ルール

- バージョンは frontmatter の `xtx_version` を使用する
- 値が0または未記入の項目は省略する
- ARA00000（会社情報）は REF タグを使用する
- 「総額①」＝「留保②」の場合（社外流出なし）は両方同じ値を設定する
- 仮計・小計・所得金額はAIが自動計算する

### タグ構造

```python
{
  "ARA00000": {
    "ARA00010": {
      "ARA00020": "REF:JIGYO_NENDO_FROM",
      "ARA00030": "REF:JIGYO_NENDO_TO"
    },
    "ARA00040": "REF:NOZEISHA_NM"
  },
  "ARB00000": {
    "ARB00010": <当期利益 総額>,
    "ARB00020": <当期利益 留保>                # 社外流出なしなら総額と同値
  },
  "ARC00000": {                               # 加算（値がある項目のみ）
    "ARC00010": {"ARC00020": <法人税>, "ARC00030": <留保>},
    "ARC00040": {"ARC00050": <道府県民税等>, "ARC00060": <留保>},
    "ARC00100": {"ARC00110": <納税充当金>, "ARC00120": <留保>},
    "ARC00130": {"ARC00140": <附帯税等>},      # 社外流出のみ
    "ARC00160": {"ARC00170": <償却超過額>, "ARC00180": <留保>},
    "ARC00190": {"ARC00200": <役員給与>},      # 社外流出のみ
    "ARC00213": {"ARC00215": <交際費等>},      # 社外流出のみ。別表十五から転記
    "ARC00220": {"ARC00230": "<区分名>", "ARC00240": <総額>, "ARC00250": <留保>},
    "ARC00325": {"ARC00330": <加算小計 総額>, "ARC00340": <加算小計 留保>}
  },
  "ARD00000": {                               # 減算（値がある項目のみ）
    "ARD00070": {"ARD00080": <受取配当益金不算入>},  # 社外流出のみ
    "ARD00100": {"ARD00110": <中間納付等>, "ARD00120": <留保>},
    "ARD00130": {"ARD00140": <所得税等還付>},  # 社外流出のみ
    "ARD00160": {"ARD00170": "<区分名>", "ARD00180": <総額>, "ARD00190": <留保>},
    "ARD00280": {"ARD00290": <減算小計 総額>, "ARD00300": <減算小計 留保>}
  },
  "ARE00000": {                               # 仮計（自動計算）
    "ARE00010": <仮計 総額>,
    "ARE00020": <仮計 留保>
  },
  "ARK00000": {"ARK00010": <合計 総額>, "ARK00020": <合計 留保>},    # AREと同値
  "ARM00000": {"ARM00010": <総計 総額>, "ARM00020": <総計 留保>},    # ARKと同値
  "ARS00000": {"ARS00010": <差引計 総額>, "ARS00020": <差引計 留保>}, # ARMと同値
  "ARS50000": {"ARS50010": <差引計 総額>, "ARS50020": <差引計 留保>}, # ARSと同値
  # 青色申告で欠損金控除がある場合のみ:
  # "ART00000": {"ART00010": <欠損金控除額>},
  "ARV00000": {                               # 所得金額（自動計算）
    "ARV00010": <所得金額 総額>,
    "ARV00020": <所得金額 留保>
  }
}
```

### 計算順序

1. 加算小計（ARC00325）= 加算各項目の合計
2. 減算小計（ARD00280）= 減算各項目の合計
3. 仮計（ARE）= ARB + 加算小計 − 減算小計
4. ARK・ARM・ARS00000・ARS50000 = AREと同値（別途加減算がない場合）
5. 所得金額（ARV）= ARS50000 − ART（欠損金控除）

### 注意事項

- 損失の場合: ARB・ARV はマイナス値
- 白色申告: ART00000 を省略する（欠損金控除不可）
- 中間申告がない場合: ARD00100 を省略する
