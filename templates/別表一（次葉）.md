## 法人税額の計算（QHB00000）

> 中小法人（資本金1億円以下）: 年800万円以下の部分は **15%**、超過分は **23.2%**

### 軽減税率部分（15%）

| タグID | 項目 | 値 |
|--------|------|----|
| QHB00020 | 事業年度月数 | |
| QHB00030 | 所得の金額（1,000円未満切捨） | |

> QHB00030 = min(所得金額, 8,000,000×月数/12) を1,000円未満切捨。
> 所得が800万以下なら全額ここに記入し、下の23.2%欄は空欄でよい。

### 23.2%部分（所得が800万超の場合のみ記入）

| タグID | 項目 | 値 |
|--------|------|----|
| QHB00040 | 超過所得（1,000円未満切捨） | |
| QHB00050 | 法人税額（超過分・23.2%） | |

> QHB00040 = 所得金額 − 8,000,000×月数/12。QHB00050 = QHB00040 × 23.2%（1円未満切捨）。

### 合計

| タグID | 項目 | 値 |
|--------|------|----|
| QHB00090 | 適用税率区分（kubun_CD） | `1` |
| QHB00100 | 法人税額（円） | |

> QHB00100 = QHB00030 × 15%（800万超の場合は＋QHB00050）（1円未満切捨）。

## XTX生成プロンプト（AIへ）

上記の記入欄に入力された値をもとに、別表一（次葉）（HOA116）のXML断片をPython辞書形式で出力してください。

### 基本ルール

- バージョンは frontmatter の `xtx_version` を使用する
- 値が0または未記入の項目は辞書から省略する
- `REF:JIGYO_NENDO_FROM` 等のREFタグは文字列でそのまま出力する
- `kubun_CD` は文字列で出力する（例: `{"kubun_CD": "1"}`）
- 所得が800万以下の場合、QHB00040/QHB00050 を省略する

### タグ構造

```python
{
  "QHA00000": {
    "QHA00010": {
      "QHA00020": "REF:JIGYO_NENDO_FROM",
      "QHA00030": "REF:JIGYO_NENDO_TO"
    },
    "QHA00040": "REF:NOZEISHA_NM"
  },
  "QHB00000": {
    "QHB00010": {                            # 軽減税率（15%）部分
      "QHB00020": <事業年度月数（整数）>,
      "QHB00030": <所得金額（円、1,000円未満切捨）>
    },
    # 所得が800万超の場合のみ追加:
    # "QHB00020": {
    #   "QHB00040": <超過所得（円、1,000円未満切捨）>,
    #   "QHB00050": <超過分法人税額（QHB00040 × 23.2%、1円未満切捨）>
    # },
    "QHB00080": {
      "QHB00090": {"kubun_CD": "1"},         # 中小法人等
      "QHB00100": <法人税額合計（円）>        # QHB00030×15% (+QHB00050)
    }
  },
  "QHC00000": {
    "QHC00010": <法人税額合計（QHB00100と同値）>,
    "QHC00030": {
      "QHC00050": <地方法人税額（QHC00010 × 10.3%、1円未満切捨）>
    }
  }
}
```

### 計算順序

1. `QHB00030` = min(所得金額, 8,000,000 × 月数/12) の1,000円未満切捨
2. `QHB00100` = QHB00030 × 15%（800万超なら + QHB00050）の1円未満切捨
3. `QHC00010` = QHB00100
4. `QHC00050` = QHC00010 × 10.3% の1円未満切捨

### 出力例（所得967,659円・12ヶ月の場合）

```python
{
  "QHA00000": {
    "QHA00010": {
      "QHA00020": "REF:JIGYO_NENDO_FROM",
      "QHA00030": "REF:JIGYO_NENDO_TO"
    },
    "QHA00040": "REF:NOZEISHA_NM"
  },
  "QHB00000": {
    "QHB00010": {
      "QHB00020": 12,
      "QHB00030": 967000        # 967,659の1,000円未満切捨
    },
    "QHB00080": {
      "QHB00090": {"kubun_CD": "1"},
      "QHB00100": 145050        # 967,000 × 15% = 145,050
    }
  },
  "QHC00000": {
    "QHC00010": 145050,
    "QHC00030": {
      "QHC00050": 14940         # 145,050 × 10.3% = 14,940.15 → 切捨
    }
  }
}
```
