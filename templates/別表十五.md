# 別表十五　交際費等の損金算入に関する明細書

> 交際費等の支出総額と損金算入限度額を計算し、損金不算入額を算出する。結果を別表四（ARC00215）に転記する。
> **交際費が0円の場合はこの別表を省略できる。**

---

## 会社情報（EGA00000）

| タグID | 項目 | 値 |
|--------|------|----|
| EGA00020 | 事業年度（自） | |
| EGA00030 | 事業年度（至） | |
| EGA00040 | 法人名 | |

---

## 支出交際費等の額

| タグID | 項目 | 値 |
|--------|------|----|
| EGB00000 | 支出交際費等の額の合計（円） | |
| EGH00000 | 支出接待飲食費損金算入基準額（円） | |

> EGH00000 = EGB00000 × 50%（1円未満切捨）。

---

## 損金算入限度額の選択（中小法人）

> 中小法人（資本金1億円以下）は①②のいずれか大きい方を選択する。
> ① 定額控除限度額: 8,000,000円 × 事業年度月数/12
> ② 飲食費の50%: EGH00000

### 事業年度月数・交際費合計（EGC00000）

| タグID | 項目 | 値 |
|--------|------|----|
| EGC00020 | 事業年度月数 | |
| EGC00030 | 交際費合計（EGB00000と同値） | |

### 損金算入限度額（EGD00000）

| タグID | 項目 | 値 |
|--------|------|----|
| EGD00005 | 適用方式（kubun_CD） | `1`=定額控除 / `2`=飲食費50% |
| EGD00020 | 損金算入限度額（円） | |

> kubun_CD=1 の場合: EGD00020 = 8,000,000 × 月数/12
> kubun_CD=2 の場合: EGD00020 = EGH00000

---

## 損金不算入額（EGE00000）

| タグID | 項目 | 値 |
|--------|------|----|
| EGE00000 | 損金不算入額（円） | |

> EGE00000 = max(0, EGB00000 − EGD00020)。別表四 ARC00215 に転記する。

---

## 支出交際費等の内訳（EGF00000）

### 飲食費内訳（EGF00010）

| タグID | 項目 | 値 |
|--------|------|----|
| EGF00020 | 飲食のために支出した費用（円） | |
| EGF00030 | うち交際費等から控除される費用（1人5,000円以下等）（円） | |
| EGF00040 | 差引交際費等の額（円） | |
| EGF00045 | (8)のうち接待飲食費の額（円） | |

> EGF00040 = EGF00020 − EGF00030。
> EGF00045 = EGF00040と同値（通常）。
> 飲食費の内訳がない場合: EGF00020=EGB00000、EGF00030=0。

### 計（EGF00100）

| タグID | 項目 | 値 |
|--------|------|----|
| EGF00110 | 支出交際費等の額 合計（円） | |
| EGF00120 | 交際費等の額から控除される費用の額 合計（円） | |
| EGF00130 | 差引交際費等の額 合計（円） | |
| EGF00140 | (8)のうち接待飲食費の額 合計（円） | |

> EGF00110 = EGB00000と同値。EGF00120 = EGF00030と同値。
> EGF00130 = EGF00040と同値。EGF00140 = EGF00045と同値。
> ⚠️ EGF00140 は損金不算入額ではなく「接待飲食費の額の合計」。

## XTX生成プロンプト（AIへ）

上記の記入欄に入力された値をもとに、別表十五（HOE200）のXML断片をPython辞書形式で出力してください。

### 基本ルール

- バージョンは `14.0` を使用する（固定）
- 値が0または未記入の項目は省略する
- REFタグは文字列でそのまま出力する
- `kubun_CD` は文字列で出力する
- **交際費が0円の場合はこの別表を出力しない**

### タグ構造

```python
{
  "EGA00000": {
    "EGA00010": {
      "EGA00020": "REF:JIGYO_NENDO_FROM",
      "EGA00030": "REF:JIGYO_NENDO_TO"
    },
    "EGA00040": "REF:NOZEISHA_NM"
  },
  "EGB00000": <支出交際費等の合計（円）>,
  "EGH00000": <支出接待飲食費損金算入基準額（EGB00000 × 50%）>,
  "EGC00000": {
    "EGC00020": <事業年度月数>,
    "EGC00030": <交際費合計（EGB00000と同値）>
  },
  "EGD00000": {
    "EGD00005": {"kubun_CD": "<1=定額控除 / 2=飲食費50%>"},
    "EGD00020": <損金算入限度額（円）>
  },
  "EGE00000": <損金不算入額（円）>,           # max(0, EGB00000 - EGD00020)
  "EGF00000": {
    "EGF00010": {                            # 飲食費内訳
      "EGF00020": <飲食のために支出した費用>,
      "EGF00030": <控除される費用（1人5,000円以下等）>,
      "EGF00040": <差引交際費等の額>,         # EGF00020 - EGF00030
      "EGF00045": <(8)のうち接待飲食費の額>   # EGF00040と同値（通常）
    },
    "EGF00100": {                            # 計
      "EGF00110": <支出交際費等の額合計>,     # EGB00000と同値
      "EGF00120": <控除される費用合計>,       # EGF00030と同値
      "EGF00130": <差引交際費等の額合計>,     # EGF00040と同値
      "EGF00140": <接待飲食費の額合計>        # EGF00045と同値（損金不算入額ではない）
    }
  }
}
```

### 計算順序

1. `EGH00000` = EGB00000 × 50%（1円未満切捨）
2. 定額控除限度額 = 8,000,000 × 月数/12
3. `EGD00020` = kubun_CD=1 なら定額控除額、kubun_CD=2 なら EGH00000
4. `EGE00000` = max(0, EGB00000 − EGD00020)
5. `EGF00040` = EGF00020 − EGF00030

### 注意事項

- EGF00140 は「接待飲食費の額の合計」（EGF00045の合計）であり、損金不算入額（EGE00000）ではない
- 飲食費の内訳がない場合: EGF00020=EGB00000、EGF00030=0、EGF00040=EGF00045=EGB00000
